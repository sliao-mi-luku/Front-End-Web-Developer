# HTML

### Text Elements

**Block element**

| Type | HTML |
| ----------- | ----------- |
| paragraphs | `<p>` |
| bullet points lists | `<ul>` |
| numbered lists | `<ol>` |
| headings | `<h1>` to `<h6>` |
| articles | `<article>` |
| sections | `<section>` |
| long quotes | `<blockquote>` |



**Inline element**

| Type | HTML |
| ----------- | ----------- |
| links | `<a>` |
| emphasized words | `<em>` |
| important words | `<strong>` |
